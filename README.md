# Database for all political spending in Tennessee state politics
# By Adam Friedman, data and investigative reporter with the Tennessee Lookout

For the Tennessee Lookout, I maintain a database called "Cash for Clout," which tracks spending on state politics across three categories. Political donations, independent expenditures and lobbying.

This project is broken down into a Jupyter Notebook, which scrapes the Tennessee Registry of Election Finance's campaign contribution and expenditures database and organizes that data. 

For lobbying spending, I make a twice-a-year request to the Registry, which provides us with a CSV covering the latest six-month period.
